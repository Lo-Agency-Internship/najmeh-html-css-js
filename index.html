HTML stands for hyper text markup language which can be used by web developers for creating a web page.

In a short form, we can say the basic structure of an HTML document includes:DOCTYPE, HTML lan, head, title, meta and body.

Tha main elements that I've learned so far are:html, head, title, body, h1/h2/... and p.

In a simple way, html relates to the root of an HTML page, head gives us some information about page, title is the title of page, body is the container of all visible tags, h1/h2/... define a large heading and p defines a paragraph.

Their important attributes include:background, color, font, img, link, class, id and so on

The attributes can be used inside the correct tags.
